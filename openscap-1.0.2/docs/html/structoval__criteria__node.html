Detailed Description

Specify oval_definition dependencies.

An Oval_criteria_node specifies oval_definition dependencies as follows:

  • if oval_criteria_node_get_type == OVAL_NODETYPE_CRITERIA - a hierarchic dependency on other Oval_criteria_nodes
  • if oval_criteria_node_get_type == OVAL_NODETYPE_CRITERION - an oval_test dependency
  • if oval_criteria_node_get_type == OVAL_NODETYPE_EXTENDDEF - an oval_definition dependency
